# FinTrack 資產管理系統 — 功能規格書

## 1. 專案概述
一套個人跨國（台灣／越南／美國／其他地區）資產、收入、支出管理系統。
核心功能：淨資產總覽、多幣別資產登記、收入登記（含頻率換算）、月度支出登記、
12個月財務趨勢分析。所有外幣金額都會依即時匯率換算成 TWD 做加總與圖表呈現。

側邊欄導覽（固定五個頁面）：
- `/` 總覽
- `/assets` 資產
- `/incomes` 收入
- `/expenses` 花費
- `/trends` 趨勢
側邊欄底部：使用者頭像/名稱/Email + 登出按鈕。

## 2. 資料模型

### Asset（資產）
| 欄位 | 型別 | 說明 |
|---|---|---|
| name | string | 資產名稱 |
| category | enum | 見下方分類（含分組） |
| region | enum | TW / VN / US / OTHER |
| amount | number | 金額（原始幣別） |
| currency | enum | TWD/VND/USD/EUR/JPY/GBP/HKD/SGD/AUD/CAD/KRW |
| note | string? | 備註（選填） |

**類別分組（下拉選單結構）：**
- 現金與存款：現金/存款(台灣)、現金/存款(越南)、現金/存款(其他)
- 投資：台股、ETF/基金、越南股票、美股
- 不動產：台灣房產、越南房產
- 保障：保險
- 勞退/退休金
- 負債：負債
- 其他：其他

### Income（收入）
| 欄位 | 型別 | 說明 |
|---|---|---|
| name | string | 收入名稱 |
| type | enum | 主動收入(薪資) / 被動收入 / 投資收益 / 其他收入 |
| region | enum | TW/VN/US/其他 |
| amount, currency | — | 同上 |
| frequency | enum | 每月 / 每年 / 單次 |
| note | string? | 備註 |

### Expense（支出）
| 欄位 | 型別 | 說明 |
|---|---|---|
| name | string | 項目名稱 |
| category | enum | 餐飲/交通/住房水電/娛樂/醫療健康/購物/工作相關/其他 |
| region | enum | TW/VN/US/其他 |
| amount, currency | — | 同上 |
| date | date | 日期（預設當天） |
| note | string? | 備註 |

### ExchangeRate（匯率快取）
定期抓取的匯率表，畫面上會顯示「匯率 HH:MM 更新」時間戳。

## 3. 商業邏輯

1. **貨幣換算**：所有加總/圖表數字皆換算為 TWD；原始清單項目仍顯示原幣別。
2. **月均收入計算**：
   - 頻率＝每月 → 直接計入月均。
   - 頻率＝每年 → 月均 = 金額 ÷ 12。
   - 頻率＝單次 → 不計入月均收入，只在收入清單顯示。
3. **本月支出**：僅加總「當月」實際登記的支出（非平均）。
4. **總淨資產公式**：`(現金+投資+不動產+勞退) − (負債+保險)`，換算為 TWD。
5. **月度明細結餘**：`結餘 = 月均收入合計(TWD) − 當月實際支出(TWD)`。
6. CRUD：每筆資料列都有「編輯（鉛筆）」「刪除（垃圾桶）」圖示按鈕。
7. 地區篩選為前端 tab 篩選，不影響底層資料。

## 4. 頁面規格

### 4.1 總覽 `/`
- 標題「財務總覽」+ 日期/地區清單/匯率更新時間
- 總淨資產卡片（大數字 + 公式說明文字）
- 三張地區資產卡（台灣/越南/美國）：旗幟+地區+幣別badge、金額、換算TWD、共N筆
- 月均收入狀況卡（綠色漸層）：TWD/VND 月均、合計
- 本月支出狀況卡（橘紅漸層）：TWD/VND 本月支出、合計
- 近6個月支出多線圖（依分類，換算TWD）
- 花費記錄卡：新增支出按鈕、月份選擇、地區篩選tab、當月清單/空狀態
- 當月分類花費趨勢卡（無資料時顯示「本月無花費」）
- 月曆助手卡：近4個月支出橫向長條圖

### 4.2 資產 `/assets`
- 新增資產按鈕（右上）
- 摘要卡：台灣資產(TWD)、越南資產(VND)、負債總計
- 篩選tab：全部/台灣/越南/美國/其他地區
- 依分類分組列表（現金與存款、投資、不動產、保障、勞退/退休金、負債、其他），
  每組顯示小計，每列含編輯/刪除按鈕
- 新增/編輯彈窗欄位：資產名稱、類別（分組select）、地區、金額、幣別、備註

### 4.3 收入 `/incomes`
- 新增收入按鈕
- 摘要卡：月均收入(TWD)含年約、月均收入(VND)含年約
- 分組列表：主動收入(薪資)、被動收入、投資收益、其他收入；
  每列顯示地區badge、頻率badge，若為「每年」則額外顯示「月均 $X」
- 新增/編輯彈窗欄位：收入名稱、類型、地區、金額、幣別、頻率、備註

### 4.4 花費 `/expenses`
- 新增花費按鈕
- 月份導航：‹ 2026年7月 ›
- 篩選tab：全部/台灣/越南/其他
- 當月清單 / 空狀態（插圖+文字+CTA按鈕）
- 新增/編輯彈窗欄位：項目名稱、類別（8選項）、地區、金額、幣別、日期（date picker）、備註

### 4.5 趨勢 `/trends`
- 圖1：月均收入 vs 支出趨勢（近12個月折線圖，TWD）
- 圖2：分類支出趨勢（近6個月堆疊長條圖，TWD）
- 表格：月度明細（月份/收入/支出/結餘），支出為0顯示「-」

## 5. 視覺風格重點
- 深色側邊欄（深藍/近黑），active導覽項目高亮藍色
- 卡片大圓角、輕陰影，白底內容區
- 色彩語意：台灣=藍、越南=紅、收入=綠漸層、支出=橘紅漸層
- 圖示：地區用國旗emoji，分類用對應icon，列表操作用鉛筆/垃圾桶小icon
- 圖表：折線圖 + 堆疊長條圖

## 6. 建議技術棧（Cloudflare 部署）
- 前端：React + Vite（或 Next.js static export）+ Tailwind CSS
- 圖表：Recharts 或 Chart.js
- API：Cloudflare Workers（建議搭配 Hono 框架）
- 資料庫：Cloudflare D1（對應上方 Asset/Income/Expense 三張表）
- 匯率來源：第三方匯率 API，用 Workers KV/Cache 做定時快取
- 認證：自建 email/password + JWT，或 Workers Sessions
- 部署：Cloudflare Pages（前端）＋ Pages Functions/Workers（API）＋ D1（資料庫）
